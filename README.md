# 📊 End-to-End Data Analytics Pipeline – Sales Analytics

## 🚀 Project Overview

This project demonstrates a complete **End-to-End Data Analytics workflow** used in real-world industry scenarios.
It covers the full pipeline from **raw data processing to business insights visualization**.

The project starts with messy Excel data, which is cleaned using Python, stored in a SQL database, and finally visualized using an interactive Power BI dashboard.

---

## 🔥 Workflow Architecture

Excel (Raw Data) ➜ Python (Data Cleaning & Transformation) ➜ MySQL (Database Storage) ➜ Power BI (Dashboard & Insights)

---

## 🧠 Key Features

* 🧹 Data Cleaning & Preprocessing using Python (Pandas)
* 🔄 ETL Pipeline Implementation (Extract, Transform, Load)
* 🗄️ MySQL Database Design & Data Storage
* 📊 Interactive Power BI Dashboard with KPIs
* 📈 Business Insights Generation (Sales, Profit, Trends)

---

## 📊 KPIs Implemented

* 💰 Total Sales
* 📈 Total Profit
* 📊 Profit Margin (%)
* 🧾 Total Orders
* 🛒 Average Order Value (AOV)
* 📉 Sales Trends (Monthly/Yearly)

---

## 🛠️ Tech Stack

* **Excel** – Raw Data Source
* **Python (Pandas, NumPy)** – Data Cleaning & Transformation
* **MySQL** – Database Management
* **Power BI** – Data Visualization & Dashboard
* **DAX** – KPI Calculations

### 2️⃣ Load Data into MySQL

* Run Python script to push cleaned data

### 3️⃣ Power BI Dashboard

* Connect Power BI to MySQL
* Load data and view dashboard
